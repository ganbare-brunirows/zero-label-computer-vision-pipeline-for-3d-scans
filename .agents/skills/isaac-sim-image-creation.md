# Isaac Sim Synthetic Data Generation Skill

## Instructions
1. This skill expects a prepared USD file (typically generated by the Houdini prep script) and the folder containing the textures for the objects in the USD file.
2. Execute the Isaac Sim python script designed to load the USD, and create and assign materials to the objects in the USD file. `isaac_sim_materials_asignator.py`
3. (You will need to ensure the Isaac Sim python environment script, usually `python.sh` or `python.bat` provided by Omniverse, is used to run the generation script).
4. Verify that the primitives have their materials assigned correctly.
5. Save the usd file and report the output path to the user.
