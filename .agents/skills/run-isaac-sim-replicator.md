# Isaac Sim Synthetic Data Generation Skill

## Instructions
1. This skill expects a prepared USD file with materials(typically generated by the isaac-sim-image-creation skill), the folder where the dataset is going to be saved (typically the one where the usd file is located)
2. Inform the user before running the script about the argument you are going to pass to the `ejecutar_replicator_randomization` script
2. Execute the Isaac Sim python script designed to load the USD, and run Replicator. 
3. Verify that the output dataset is created in the specified output directory and contains the required annotations (e.g., YOLO format bounding boxes) and images.
4. If there are memory issues or Omniverse errors, report them to the user.
